# Program Hitung Faktorial
Program Hitung Faktorial digunakan untuk menghitung faktorial dari sebuah bilangan bulat positif yang dimasukkan oleh pengguna.
CARA KERJA:
Program menggunakan Scanner untuk membaca input dari keyboard.
Bilangan disimpan dalam variabel n.
Jika n < 0, program menolak menghitung faktorial.
Jika n >= 0, program menghitung faktorial dengan perulangan:
Faktorial dihitung dengan rumus: n!=1\times 2\times 3\times \dots \times n- Hasil faktorial ditampilkan ke layar.
